> Curated list of ESP32 security tools, firmware, and resources for penetration testers and IoT researchers.

## Categories

### Firmware Analysis
- **esptool.py** — ESP32 firmware flash/dump tool
- **Binwalk** — Firmware extraction and analysis
- **FACT** — Firmware Analysis and Comparison Tool

### WiFi & BLE Attacks
- **ESP32 WiFi Deauther** — WiFi deauthentication attacks
- **Bluetooth Explorer** — BLE sniffing and injection
- **WiFi Pineapple** — Rogue AP frameworks

### Hardware Hacking
- **JTAG/UART** — Debug interface exploitation
- **Bus Pirate** — Protocol analysis
- **Logic Analyzers** — Signal capture and decoding

### Tools & Frameworks
- **ESP-IDF** — Official development framework
- **PlatformIO** — Cross-platform build system
- **Micropython** — Python for microcontrollers
